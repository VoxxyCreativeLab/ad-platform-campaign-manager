# PMax Asset Requirements

## Text Assets

| Asset | Min | Max | Character Limit |
|-------|-----|-----|-----------------|
| Headlines | 3 | 15 | 30 chars each |
| Long headlines | 1 | 5 | 90 chars each |
| Descriptions | 2 | 5 | 90 chars each (first can be 60) |
| Business name | 1 | 1 | 25 chars |
| Call to action | Auto | — | Selected from list or "Automated" |
| Display URL path | 0 | 2 | 15 chars each |

### Text Best Practices
- **Headlines:** Include keywords, value props, brand name, CTAs. Don't repeat across headlines.
- **Long headlines:** More descriptive — include full value proposition.
- **Descriptions:** Expand on headlines. Include features, benefits, social proof.
- **Pin sparingly** — pinning limits Google's testing ability; only pin when legally required.

## Image Assets

| Asset | Min | Recommended | Specs |
|-------|-----|-------------|-------|
| Landscape image | 1 | 4+ | 1200×628 (min 600×314), 1.91:1 ratio |
| Square image | 1 | 4+ | 1200×1200 (min 300×300), 1:1 ratio |
| Portrait image | 0 | 2+ | 960×1200 (min 480×600), 4:5 ratio |
| Square logo | 1 | 1 | 1200×1200 (min 128×128), 1:1 ratio |
| Landscape logo | 0 | 1 | 1200×300 (min 512×128), 4:1 ratio |

### Image Best Practices
- **Max file size:** 5120 KB
- **Formats:** JPG, PNG (no GIF, no animation)
- **Safe area:** Keep important content in center 80% (edges may be cropped)
- **No text overlays** on images (text is added dynamically by Google)
- **No logos on images** (logo is a separate asset)
- **High quality:** No blurry, pixelated, or heavily filtered images
- **Diverse subjects:** Show different products, angles, use cases
- **Lifestyle + product shots:** Mix pure product images with in-context lifestyle shots

## Video Assets

| Requirement | Spec |
|-------------|------|
| Min duration | 10 seconds |
| Recommended | 15-30 seconds |
| Orientation | Landscape (16:9), Square (1:1), or Vertical (9:16) |
| Hosting | Must be on YouTube (public or unlisted) |
| Min recommended | 1 per asset group |

### Video Best Practices
- **If you don't provide video,** Google auto-generates one from images and text (quality varies)
- **Always provide at least one custom video** — auto-generated videos underperform
- **Include multiple orientations** — landscape for YouTube, vertical for Shorts/mobile
- **Hook in first 5 seconds** — many placements are skippable
- **Include CTA** in the video and as overlay

## Asset Group Strategy

### Single-Product Business
One asset group with all assets focused on the core offering.

### Multi-Product E-commerce
Create separate asset groups per product category:
```
Asset Group: Running Shoes
  └── Audience signal: In-market for running shoes
  └── Listing group: Running shoes category
  └── Assets: Running shoe images, running-specific headlines

Asset Group: Hiking Boots
  └── Audience signal: In-market for hiking gear
  └── Listing group: Hiking boots category
  └── Assets: Hiking boot images, hiking-specific headlines
```

### Asset Performance Ratings
Google rates each asset: **Best**, **Good**, **Low**, or **Learning**.
- Replace "Low" performing assets after 2-4 weeks
- Keep "Best" and "Good" assets
- Assets in "Learning" need more data — wait before judging
- Minimum 2-4 weeks of data before making changes

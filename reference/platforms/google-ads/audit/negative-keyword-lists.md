## Usage Instructions

1. Go to Google Ads → Tools → Shared library → Negative keyword lists
2. Create a new list for each category
3. Add the relevant keywords from above
4. Apply lists to appropriate campaigns
5. Review and expand quarterly based on search term reports
6. Don't over-negate — check if negatives are blocking valuable traffic
